## What is Qafka?

Qafka is a modern, universal queue management platform designed to handle message queues across different backends with a unified API. Built with Go and React, it provides a scalable solution for managing message queues, workers, and monitoring in distributed systems.

## Quick Start

Get up and running with Qafka in minutes:

<CodeGroup>

```bash Terminal
# Clone the repository
git clone https://github.com/shravan20/qafka.git
cd qafka

# Run the setup script
chmod +x scripts/setup.sh
./scripts/setup.sh

# Start the application
docker-compose up -d
```

```bash Manual Setup
# Start PostgreSQL
docker-compose -f docker-compose.dev.yml up -d

# Start the backend
cd backend && go run cmd/main.go

# Start the frontend (in another terminal)
cd frontend && npm run dev
```

</CodeGroup>

## Architecture

Qafka follows a modern microservices architecture:

- **Backend**: Go with Fuego framework for high-performance REST API
- **Frontend**: React with Vite and TypeScript for modern UI
- **Database**: PostgreSQL with Bun ORM for reliable data storage
- **Monitoring**: Prometheus for metrics collection
- **Plugins**: Go-based plugin system for extensibility
